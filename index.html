Features

Everything you need,
nothing you don't

Built with simplicity and power in mind. GitHub Store gives you a first-class experience for discovering and installing open source apps.

Smart Search

Search across millions of GitHub repositories. Filter by platform — Android, Desktop, Linux, iOS — and discover trending projects.

One-Click Install

Download APK files directly from GitHub releases with a single tap. No need to browse the web — everything is right in the app.

Favorites

Save your favorite repositories for quick access later. Your curated collection of must-have open source apps, always at hand.

Dark & Light Themes

Full Material You support with system-auto, dark, and light modes. The app adapts beautifully to your preference.

Multi-Language

Full English and Russian localization out of the box. Clean, native strings — no machine translations.

Works Without Login

No account needed. Use the app immediately with GitHub's public API (60 req/hr). Add a token for 5,000 req/hr.
